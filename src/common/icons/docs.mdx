## Как использовать

1. Все иконки имеют фиксированные размеры и не должны быть масштабируемы при обычных обстоятельствах.
2. При необходимости вы можете настроить компонент как обычное SVG изображение, передавая пропсы SVG.
3. Цвет иконки задается параметром `currentColor` (в большинстве случаев).
