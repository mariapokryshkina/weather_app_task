] );

Набор цветов

> Если вы **изменили имена** переменных, **удалили** старые или **добавили** новые, вам необходимо изменить файл `common/styles/export/colors.module.scss` или автоматически его **сгенерировать**. Для этого запустите скрипт:

<ColorPalette>
	{colors.other
		.map( ( [name, value] ) => (
			<ColorItem
				title={name}
				key={name}
				colors={{
					[`$${name}`]: value,
				}}
			/>
	) )}
	<ColorItem
		title="gray"
		colors={
			Object.fromEntries(
				colors.gray
					.map( ( color ) => ( [
						`$${color[0]}`,
						color[1]
					] ) )
			)
		}
	/>
</ColorPalette>
